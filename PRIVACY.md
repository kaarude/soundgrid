# Privacy

SoundGrid processes microphone and audio data locally. It does not include
analytics, usage telemetry, advertising, accounts, or cloud storage.

When passthrough is enabled, SoundGrid captures the selected physical
microphone and mixes it into the selected virtual mic output. Audio is not
recorded or transmitted by SoundGrid. Imported clips and settings remain in
the operating system's local application-data directory.

SoundGrid contacts GitHub to check for application updates and contacts the
VB-Audio website only when the user opens its donation page or when the build
process downloads the pinned driver package. Voice applications receiving the
virtual microphone output have their own privacy behavior and policies.
